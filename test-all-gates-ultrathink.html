  • 自動で周期的に出力が切り替わる

🔘 INPUTスイッチ

  • ダブルクリックでON/OFF切り替え

💡 OUTPUT表示

  • 入力が0の時: 💡が暗い

CUSTOM

🔧 カスタムゲート

作成手順:

  1. 回路を作成（INPUT/OUTPUT必須）
  2. 「回路→IC」ボタンをクリック
  3. 名前とアイコンを設定
  4. ツールパレットから使用可能

特徴:

  • 複数入力・複数出力対応
  • 紫色の特別な外観
  • LocalStorageに自動保存
  • 階層的設計が可能

🎮 操作ガイド

基本操作:

  • ゲート配置: ツールパレットから選択 → キャンバスをクリック
  • ゲート移動: ドラッグ＆ドロップ
  • ワイヤー接続: ピンクリック → 接続先ピンクリック
  • ワイヤー削除: ワイヤーを右クリック
  • ゲート削除: ゲートを選択 → Deleteキー

ショートカット:

  • Ctrl+Z / Cmd+Z: 元に戻す
  • Ctrl+Y / Cmd+Y: やり直し
  • Escape: ワイヤー描画キャンセル
