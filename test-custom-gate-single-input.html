🔧 カスタムゲート 1入力1出力テスト

💡 ヒント: ワイヤー（接続線）を削除するには、ワイヤーを右クリックしてください！

📋 テストケース: 1入力1出力のバッファーゲート作成

手順

1. キャンバスをクリア
全てのゲートとワイヤーを削除
2. シンプルな回路を作成
- INPUTゲートを1つ配置
- OUTPUTゲートを1つ配置
- INPUTからOUTPUTへワイヤーで接続
3. カスタムゲートに変換
「回路→IC」ボタンをクリック
4. ダイアログを確認
期待結果:
- 入力ピン: 1個（A）
- 出力ピン: 1個（Y）
✅ 既に修正済み！initialInputs/initialOutputsが正しく反映されるようになった
5. カスタムゲートを作成
名前を入力して「作成」ボタンをクリック
6. 作成されたカスタムゲートを配置
ツールパレットから作成したカスタムゲートをクリック
7. 表示を確認
期待結果:
- カスタムゲートの入力ピン: 1個
- カスタムゲートの出力ピン: 1個

🔍 カスタムゲート出力テスト

手順

1. カスタムゲートに入力を接続
- INPUTゲートを配置
- INPUTをON（緑）にする（ダブルクリック）
- INPUTとカスタムゲートをワイヤーで接続
2. カスタムゲートの出力を確認
期待結果:
- カスタムゲートの出力ピンが緑色になる
- 接続されたワイヤーも緑色になる
3. OUTPUTゲートを接続
- OUTPUTゲートを配置
- カスタムゲートとOUTPUTをワイヤーで接続
期待結果:
- OUTPUTゲートが光る（💡が明るくなる）

🎯 複数入出力テスト

手順

1. 3入力2出力の回路を作成
- INPUTゲートを3つ配置（A, B, C）
- ANDゲートとORゲートを配置
- A,BをANDゲートに接続
- B,CをORゲートに接続
- OUTPUTゲートを2つ配置
- ANDの出力を1つ目のOUTPUTに接続
- ORの出力を2つ目のOUTPUTに接続
2. カスタムゲートに変換
「回路→IC」ボタンをクリック
期待結果:
- 入力ピン: 3個（A, B, C）
- 出力ピン: 2個（Y, O1）
🚀 デバッグのヒント:
- ブラウザのコンソールを開いてログを確認
- 右クリックでワイヤーを削除できる
- キーボードショートカット: Ctrl+Z（元に戻す）、Ctrl+Y（やり直し）
