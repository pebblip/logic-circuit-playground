🔧 BINARY_COUNTERゲート配置修正テスト

🎯 テスト手順

1. アプリを起動: npm run dev でアプリケーションを起動
2. ツールパレットから「COUNTER」をドラッグ: 左サイドバーの「特殊ゲート」セクションからCOUNTERボタンを見つける
3. キャンバスにドロップ: COUNTERをキャンバス中央部分にドラッグ&ドロップ
4. ブラウザコンソールを確認: F12でデベロッパーツールを開き、Consoleタブで以下のログを確認

🎯 標準ゲート配置実行: {type: "BINARY_COUNTER", position: {x: ..., y: ...}}

- BINARY_COUNTERゲートが表示される
- ラベル: "COUNTER"、"2bit"、値: "00"
- CLK入力ピンと2つの出力ピンが表示される

❌ 修正前の問題（参考）

修正前のエラーログ:

(または座標変換に関連するエラー)
