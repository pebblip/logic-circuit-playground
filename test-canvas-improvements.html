🧪 キャンバス改善機能テスト

新しいUX機能が正しく動作するかテストしましょう！

⚡ クイックテスト（1分で完了）

以下の3つをすぐに試してみてください：

  1. ツールパレットから ANDゲート をドラッグしてキャンバスに配置
  2. Space を押しながらキャンバスをドラッグして移動
  3. 空白エリアでドラッグして複数ゲートを選択

🎨 ドラッグ＆ドロップ配置

テスト手順

  1. 左側のツールパレットから INPUT ゲートをドラッグ開始
  2. 半透明のプレビューが表示されることを確認
  3. キャンバスの好きな位置でドロップ
  4. 異なる種類のゲート（AND、OR、NOT）でも試す
  5. カスタムゲートでも動作することを確認

✅ 成功条件

  • ドラッグ中に半透明プレビューが表示される
  • ドロップした位置にゲートが配置される
  • 従来のクリック配置も引き続き動作する

🗺️ 無限キャンバス移動

テスト手順

  1. Space キーを押し続ける
  2. カーソルが手のひらマーク（grab）に変わることを確認
  3. マウスをドラッグしてキャンバスを移動
  4. 別の方法も試す：
    • マウスの中ボタンでドラッグ
    • Ctrl + 左クリックでドラッグ

✅ 成功条件

  • Google Mapsのようにスムーズに移動できる
  • ゲートの配置に影響しない
  • ワイヤー描画中はパンが無効になる

🎯 複数選択と一括操作

準備

まず、5〜6個のゲートを配置してください

テスト手順

  1. 空白エリアでマウスをドラッグ（選択矩形が表示）
  2. 複数のゲートが青い枠で選択される
  3. Shift + クリックで選択に追加
  4. 選択したゲートをドラッグして一括移動
  5. Delete キーで一括削除

✅ 成功条件

  • 選択されたゲートが青い枠（#00aaff）で表示
  • 複数ゲートが一緒に移動する
  • Deleteキーで選択したゲートが全て削除される
  • Undo（Ctrl+Z）で復元できる

🚀 統合動作テスト

実践的なワークフロー

  1. 半加算器回路を作成：
    • INPUTを2つドラッグ配置
    • XORゲートとANDゲートをドラッグ配置
    • OUTPUTを2つドラッグ配置
  2. 配置が狭い場合、Space+ドラッグでビューを調整
  3. ワイヤーで接続（ピンクリック）
  4. 回路全体を選択（ドラッグ選択）
  5. 選択した回路を別の場所に移動

✅ 成功条件

  • ドラッグ配置で素早く回路構築
  • パン機能で作業領域を自由に移動
  • 回路全体を簡単に移動できる

🔍 エッジケーステスト

特殊な状況での動作確認

  1. ワイヤー描画中：
    • ピンをクリックしてワイヤー描画開始
    • この状態でSpaceキーを押してもパンしない
    • ESCキーでワイヤー描画をキャンセル
  2. 重なったゲート：
    • ゲートを重ねて配置
    • 上のゲートだけが選択される
  3. 大量選択：
    • 20個以上のゲートを配置
    • 全て選択して移動（パフォーマンス確認）

⌨️ キーボードショートカット

全ショートカットの動作確認

Space + ドラッグ パン
Delete 削除
Ctrl+Z 元に戻す
Ctrl+Y やり直し
Shift+クリック 選択追加
ESC キャンセル
