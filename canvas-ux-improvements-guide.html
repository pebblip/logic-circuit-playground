🎯 複数選択＆一括操作 NEW

複数のゲートを選択して、まとめて移動・削除できます！

選択方法

  • 空白エリアでドラッグ → 範囲選択
  • Shift + クリック → 選択に追加
  • Ctrl/Cmd + クリック → 選択の切り替え

一括操作

  • 選択したゲートをドラッグ → 一括移動
  • Delete / Backspace → 一括削除

デモシナリオ

  1. 複数のゲートを配置
  2. ドラッグで範囲選択（青い枠で囲まれる）
  3. 選択したゲートをまとめて移動
  4. 不要なゲートを一括削除
